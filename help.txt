
Available Options:

*Provide a one-word search term

	Result:
		List of artwork from The UMMA Exchange that matches that search term.


<[Number]> + <Browser>
	
	Result:
		A browser window will open with The UMMA Exchange page for the object 		that corresponds with the number you entered. 


<[Number]> + <Artist>
	
	Result:
		Information about the artist of the object that corresponds with the 		number you entered will be returned.
 
<Artists>
	
	Result:
		A browser window will open with a bar graph of all artists represented 		in a query result.

<Medium>

	Result:
		A browser window will open with a pie chart of the different media 			represented in a query result.
